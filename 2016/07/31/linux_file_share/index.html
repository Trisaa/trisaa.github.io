虚拟机与主机共享文件夹的创建

  • Linux
  • Windows
  • 共享文件夹
studying

物理环境：主机Ubuntu14.04,虚拟机Virtualbox,虚拟机中的主机win10
为什么要写：网络上大多数共享文件夹的创建教程基本都是windows为主机，linux为虚拟主机，自己花了点时间折腾了下这个逆过程。
1.打开虚拟机，首先安装Virtualbox增强功能包（点击Devices->Insert Guest Additions CD image）。
2.点击状态栏“设备->共享文件夹”，点击右上角添加按钮，选择共享文件夹路径，共享文件夹名称，如ShareFolder，选择固定分配，确定即可。
3.进入到win10系统中，找到网络映射驱动器 Win10网络映射驱动器 ，驱动器盘符自选，文件夹填“\vboxsvr\你的共享文件夹名”，如我的“\vboxsvr\ShareFolder”,其他选项默认，点击完成。（第一次创建可能会报错，不用担心）
4.在win10我的电脑-设备和驱动器就能看到VirtualBox Guest Additions,双击进行安装。完成安装后再执行步骤三，这次就能创建成功，然后你会看到多出来一个叫“ShareFolder”的盘，这时win10中这个盘跟主机的ShareFolder文件夹就能共享文件了，It’s cool!
